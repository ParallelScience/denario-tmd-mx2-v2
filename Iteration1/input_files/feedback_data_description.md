The dataset description explains the observed limitations in the research results, specifically regarding the missing elastic property data. The dataset contains elastic constants for only 90 of the 202 materials because the Materials Project does not perform these computationally expensive calculations for metastable structures. This constraint directly necessitated the use of a secondary predictive model (Random Forest regressor) to estimate elastic properties for the 112 metastable candidates, as well as the reliance on a proxy classification (is_robust) rather than direct experimental or DFT-computed values for the entire population. Additionally, the exclusion of 16 materials from the DOS analysis due to missing data in the source database is explicitly addressed by the preprocessing plan, confirming that the analysis results are bounded by the availability of data in the Materials Project.
