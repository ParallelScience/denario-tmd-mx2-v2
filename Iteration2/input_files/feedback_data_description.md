The dataset description explains a significant limitation that directly impacts the analysis: the elastic properties (`K_vrh`, `G_vrh`, etc.) are only available for 90 out of 202 materials because the Materials Project does not perform these calculations for metastable structures. This constraint forces the research plan to rely on a subset of the data for training the robustness classifier and necessitates the use of imputation and OOD analysis to handle the 112 metastable candidates. Consequently, the reliability of the "mechanical robustness" predictions for the metastable population is inherently limited by the lack of ground-truth elastic data for those specific materials, as the model must extrapolate from the stable/near-stable subset.
